# Ejercicio 1

Crear repo local:

En Shell ejecutar

$git init .



Ejecutar git status:

$git status


Agregar archivo a git

$git add README.md


Configurar identificación del desarrollador:

 $git config --global user.email "jonathan.carrasco.garcia@gmail.com"
 $git config --global user.name "Jonathan Carrasco"


 Después ejecutar git status


 $git status
